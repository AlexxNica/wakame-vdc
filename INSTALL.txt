
Create database

mysql -uroot
> create database wakame_dcmgr default character set utf8;
> grant all on wakame_dcmgr.* to wakame_dcmgr@localhost identified by 'passwd';

# for test
> create database wakame_dcmgr_test default character set utf8;
> grant all on wakame_dcmgr_test.* to dcmgr_test@localhost identified by 'passwd';

> exit

Make sure to be installed rubygems (>= 1.3.5) and bundler (>= 0.7.0).

Setup with the development gems. i.e. shotgun, rspec
$ gem bundle --only development
Setup only with runtime gems.
$ gem bundle

"gem bundle" will make vendor/ and bin/ folders.

start up
$ rake run

spec
$ rake spec
