Wakame-VDC
============

Wakame-VDC is a toolkit for IaaS cloud.

See details for http://wakame.jp/ (English: http://en.wakame.jp/)

Features:

* Operation
** Web GUI
** RESTful API
* Hypervisor
** KVM
** LXC
* Network
** Security Group (L3)
** Distributed Firewall
** Distributed NAT
* Storage
** Solaris ZFS + iSCSI
** Tired snapshot management
* Management
** Per Account Quota


Required Components
--------------------

* RabbitMQ
* MySQL
* nginx
* Ruby (>= 1.8.7)
